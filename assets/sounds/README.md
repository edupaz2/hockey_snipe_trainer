# Sound Assets Placeholder

Place your sound files in this directory:

- `hit.mp3` - Sound when target is hit
- `miss.mp3` - Sound when player misses
- `countdown.mp3` - Countdown beep (3, 2, 1)
- `game_start.mp3` - Game start sound
- `game_end.mp3` - Game over sound
- `click.mp3` - Button click sound
- `target_on.mp3` - Target activation sound
- `streak.mp3` - Combo streak sound (every 5 hits)
- `high_score.mp3` - New high score celebration
- `error.mp3` - Error sound

## Recommended Free Sound Sources

- [Freesound.org](https://freesound.org)
- [OpenGameArt.org](https://opengameart.org)
- [Zapsplat.com](https://www.zapsplat.com)

## Format

MP3 files are recommended for compatibility across platforms.
Keep files small (< 100KB each) for quick loading.
